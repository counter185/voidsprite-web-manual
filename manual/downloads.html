◆Automatic builds

These versions are served by nightly.link and are automatically built whenever a change to the code is made. They contain the latest features, but may be unstable.

  • ◆Windows (x64) [for most PCs] voidsprite-build-win-x64.zip
  • ◆Windows (x86) voidsprite-build-win-x86.zip
  • ◆Windows (arm64) voidsprite-build-win-ARM64.zip
  • ◆Linux (AppImage) voidsprite-build-linux-appimage-x64.zip
  • ◆Linux (Flatpak) voidsprite-build-linux64-flatpak-x86_64.flatpak.zip
  • ◆macOS (x64, arm64) voidsprite-build-mac-universal.zip
  • ◆Android (arm64-v8a, armeabi-v7, x64) voidsprite-build-android.zip

void◆sprite is currently considered alpha software.
Some features may be missing or not work.

◆Monthly builds

These versions are built and released at the end of every month. They don't have the latest features, but may be more stable.

Monthly builds are available on:

  • itch.io: cntrpl.itch.io/voidsprite
  • GitHub: github.com/counter185/voidsprite/releases

◆Mac note

If you're getting the error:
Apple could not verify that "voidsprite.app" is free of malware that may harm your Mac or compromise your privacy.
Run this command in Terminal:

xattr -d com.apple.quarantine voidsprite.app

◆void◆sprite autoupdater

void◆sprite autoupdater is available through itch.io for supporters who donate $2 or more.

The autoupdater is a launcher for void◆sprite that automatically updates it to the latest automatic build.

void◆sprite autoupdater is only compatible with x64 versions of Windows 7 and later.

◆Hosting the web version

The latest automatic build of the web version is also available through nightly.link: voidsprite-build-emscripten.zip.

index.html contains a page that launches void◆sprite, you may either modify it directly or embed it with an iframe.

The web version of void◆sprite can only be loaded through a HTTP server.
Running the page from a local file:/// URL will not work.
