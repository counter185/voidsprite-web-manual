◆Visual configurations

Visual configurations let you change the appearance of voidsprite.

◆Install a visual configuration

To install an already prepared visual configuration, copy its folder into the visualconfigs/ directory inside the Application data directory.

It should then be selectable in File > Preferences > Visual > Visual configuration.

◆Create a visual configuration

A visual configuration folder should contain a file named config.json. This file needs to be a JSON containing keys and values that define the appearance.
A sample file with all possible configurable values for the current version is generated automatically when voidsprite is launched and is saved to visualconfigs/sample_config.json, you may copy it and modify to create your own theme.

All color values in the config file must be in hexadecimal AARRGGBB format.

◆Fill values

Values starting with fill: can have the following formats:

  • fill:solid:[color] - solid color fill
  • fill:gradient:[color_top_left]/[color_top_right]/[color_bottom_left]/[color_bottom_right] - gradient fill
  • fill:texture:[file_name] - texture fill

◆Replacing assets

Default assets such as icons can be replaced by placing them in your visual configuration's folder.
The file names must match original ones, for example to replace the notif_success.png icon, place a file with the same name in your visual configuration folder.

◆Changing the font

To change the main font, place its TTF file inside the visual configuration folder.
Then, in the config.json file, set the general/font key to the name of your font file. 

{
    "general": {
        "font": "font-file.ttf"
    },
    //...
}
