◆Zlib compression

voidsprite can compress and decompress any file with zlib.

◆Decompress a zlib-compressed file

To decompress a file with zlib, drag a file with a .zlib extension into the voidsprite Launchpad.

The file will be decompressed and saved with to the same directory with the same name + the .unzlib extension.

Zlib-compressed data starts with the byte 0x78 (x in ASCII), followed by a byte indicating the level of compression.
The file you drag into the Launchpad must contain only the compressed data.

◆Compress a file

To compress a file with zlib, drag a file with an .unzlib extension into the voidsprite Launchpad.

The file will be compressed and saved with to the same directory with the same name + the .zlib extension.
