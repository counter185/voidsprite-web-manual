◆Recovery autosaves

voidsprite can automatically create periodic backups of your projects, known as Recovery autosaves.

◆Creating Recovery autosaves

◆Automatic autosaves

A Recovery autosave will be created when there are unsaved changes and a set time interval has passed.
This time interval is set to 20 minutes by default and can be changed or disabled in Preferences.

◆Manual autosaves

A Recovery autosave can also be manually triggered with File > Force recovery autosave on the navigation bar, or the Force recovery autosave keybind (default: Ctrl+Shift+R).

◆Accessing Recovery autosaves

From the voidsprite Launchpad, select File > Recovery autosaves on the navigation bar.
This will open a list of all Recovery autosaves that have been created. Click on one to open it in a new workspace.

These saved files can also be accessed directly from the autosaves/ folder in the Application data directory.
