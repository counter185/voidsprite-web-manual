◆Introduction

void◆sprite is a pixel art editor for creating sprites, spritesheets, tilesets, pixel illustrations, and anything else that pixel art is suitable for.
It is designed to be quick and simple to use, while providing advanced features for game development and romhacking.

◆Download links

◆Recommended: nightly.link (latest automatic builds): https://nightly.link/counter185/voidsprite/workflows/msbuild/main

◆itch.io (monthly releases): https://cntrpl.itch.io/voidsprite
◆GitHub releases (monthly releases): https://github.com/counter185/voidsprite/releases

void◆sprite is currently considered alpha software.
Some features may be missing or not work.
